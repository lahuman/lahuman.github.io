29 Sep 2018

Reading time ~1 minute

Excel을 이용한 문장 길찾기 게임 만들기

아내가 일일이 손으로 문장 길찾기 게임을 만들고 있는 것을 보았다.

간단하게 프로그램으로 만들어 주기로 했는데 곰곰히 생각해보니 Excel 이면 충분할 것같아서 만들어 보았다.

VBA를 이용하여 다음과 같은 모듈을 만들었다.

10진수 한글의 유니코드 범위를 입력 받아 한글 문자열로 반환 한다.

VBA 코드는 다음과 같다.

Function getKoreaWord(src As Long) As String
Dim uCode As String
uCode = Hex(src)
getKoreaWord = ChrW("&H" & uCode)
End Function

한글의 유니코드 범위는 44032~57263 임으로 다음과 같이 사용한다.

=getKoreaWord(RANDBETWEEN(44032,57263))
