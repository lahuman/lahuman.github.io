hp-15s-eq1156au 노트북 리뷰

21 Jan 2021

Reading time ~1 minute

구매는 2020년 11월 11일에 했고, 1월 20일에 제품을 받았습니다.

구매 당시 금액은 11절 이벤트로 54만원에 구매했었습니다. 현재는 판매 종료이고 가격도 60만원 초반입니다.

구매 링크 : http://www.11st.co.kr/products/3149672026

장점

  • 윈도우 10이 포함 되어 있음
  • 기본 8G 메모리 제공 + 메모리 슬롯 여분 1개
    • 2666HZ이라고 했는데, 실제로는 3200HZ 삼성 메모리가 장착 되어 있었음
  • HP 서비스 센터 이용가능(as시 - 컴알못에겐 나쁘지 않은 선택지)
  • 숫자 키보드 제공(이건 사람마다 호불호 갈림)
  • 사운드가 크고 괜찮음
  • 갓성비를 자랑함(54만원으로 구매시)

단점

  • 키보드 울림 있음(09POILKJ,M - 요 키를 좀 쎄게 누르면 울렁됨)
  • 이동키 좌우 / 위아래 비대칭
  • 메모리 증설이 어려움(뚜껑 열기 어려움)
    • 개인이 하다가 실패한 기스 내고 안쪽 걸쇠를 깨먹은 사람이 있음
    • HP 서비스 센터 이용시 36,300원 공임비 필요
  • 제품 설명서에는 지문 인식이 있는데, 실제로 없음(설명서 보고 좋다가 말았음)
  • 초기 윈도우 업데이트시 소리 안나는 문제 있음
    • 윈도우 초기화 진행으로 해결 가능

총평

키보드 울림이 가장 마음에 안들지만, 가격이 모든 것을 용서한다.

10점 만점에 8점 박습니다!
