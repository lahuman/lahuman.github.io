Gmail SMTP 설정 가이드

23 Apr 2024

Reading time ~1 minute

2024년 기준, Gmail SMTP 설정 가이드

2010년에 Gmail SMTP 설정을 간단하게 했던 기억이 있으신가요? 2022년 기준으로 설정 방식이 변경되었고, 다소 복잡해졌습니다. 이 글을 통해 2024년 기준 Gmail SMTP 설정 방법을 단계별로 알려드리겠습니다.

GMAIL 설정 진행

  • GMAIL > 톱니바퀴 > 모든 설정 보기

  • 설정 > 전달 및 POP/IMAP

APP PASSWORD 추가

이 메뉴가 검색시 계정 > 보안 > 2단계 인증에 있다고 되어 있지만, 실제로 보이지 않고 주소 입력을 통해 접근해야 합니다.

  • https://myaccount.google.com/security 에서 2-step verification 밑에 app password 메뉴가 존재 하지 않습니다.

  • https://myaccount.google.com/apppasswords 로 직접 접근해서 추가 하면 됩니다.

이렇게 수정하고 나면 GMAIL SMTP를 사용할 수 있습니다.

다음은 사용 예제 입니다.

위 화면에서

  • username : xxx@gmail.com
  • password : app password

이렇게 설정 하면 됩니다.

참고자료

  • No “App passwords” under “Security” -> “Signing in to Google” panel

  • Google - Gmail SMTP 사용을 위한 세팅
