PPT 텍스트-이미지 자동 변환기 v1.1

배포용 PowerPoint에서 폰트 깨짐과 줄바꿈 오류를 줄이기 위해, 슬라이드 안의 텍스트를 이미지로 변환해 레이아웃을 더 안정적으로 유지하는 Windows용 데스크톱 도구입니다.

프로젝트 소개

PPT를 다른 PC에서 열었을 때 가장 자주 생기는 문제는 폰트가 바뀌거나 줄바꿈이 달라지면서 전체 레이아웃이 무너지는 것입니다.
이 프로젝트는 바로 그 문제를 줄이기 위해 만들었습니다.

이 도구는 실제 Microsoft PowerPoint 엔진을 사용해 슬라이드 안의 텍스트를 이미지로 변환한 뒤, 원래 위치에 다시 배치합니다.
덕분에 발표용, 고객사 전달용, 배포용 PPT를 더 안정적으로 준비할 수 있습니다.

어떤 문제를 해결하나요?

  • 상대방 PC에 같은 글꼴이 없어도 화면 차이를 줄일 수 있습니다.
  • 줄바꿈과 텍스트 박스 레이아웃이 달라지는 문제를 줄일 수 있습니다.
  • 발표 직전에 다른 PC에서 급하게 수정해야 하는 상황을 줄일 수 있습니다.
  • 작업용 원본과 배포용 결과물을 분리하는 흐름에 잘 맞습니다.

핵심 기능

  • 자동 그룹 해제: 슬라이드 안의 그룹을 해제해 그룹 내부 텍스트까지 빠짐없이 변환합니다.
  • 정밀 위치 보정: 텍스트 오버행을 계산해 원본과 최대한 같은 위치와 크기로 복원합니다.
  • 회전값 유지: 회전된 텍스트 박스도 이미지 변환 후 원래 각도를 유지합니다.
  • 실시간 진행 표시: 현재 진행 상태와 진행률을 UI에서 바로 확인할 수 있습니다.
  • 한국어/영어 UI 지원: 한영 전환이 가능해 팀 단위 사용에도 부담이 적습니다.
  • 사전 확인 안내: 변환 전 PowerPoint 종료 여부와 파일 잠금 상태를 확인할 수 있습니다.
  • 상황별 오류 안내: PowerPoint 미설치, 파일 사용 중, 저장 실패 등을 이해하기 쉬운 팝업으로 안내합니다.

v1.1에서 좋아진 점

  • 데스크톱 UI를 더 보기 쉽게 정리했습니다.
  • 진행 상태 텍스트와 진행 바가 실제 변환 과정에서 정상적으로 보이도록 수정했습니다.
  • 변환 시작 전 확인 팝업과 주의 사항 패널을 추가했습니다.
  • 긴 파일명을 선택해도 진행 영역이 가려지지 않도록 레이아웃을 보완했습니다.
  • 그룹 내부 텍스트 처리 흐름을 개선해 변환 안정성을 높였습니다.
  • 오류 메시지를 더 사용자 친화적으로 바꿔, 실패 원인과 다음 조치를 쉽게 이해할 수 있도록 했습니다.
  • 소개 페이지와 문서를 최신 UI 기준으로 함께 정리했습니다.

실제 사용 흐름

  1. 프로그램을 실행합니다.
  2. 변환할 PPT 파일을 선택합니다.
  3. PowerPoint 종료 여부와 파일 잠금 상태를 확인합니다.
  4. 진행 상태와 진행률을 보면서 변환이 끝날 때까지 기다립니다.
  5. 원본 파일과 같은 폴더에 변환 결과 파일이 생성됩니다.

사용자 친화적 안내도 강화했습니다

이전에는 오류가 발생했을 때 일반적인 실패 메시지로 보일 수 있는 부분이 있었습니다.
지금은 실제 사용자가 자주 겪는 상황을 기준으로 더 명확하게 안내합니다.

  • Microsoft PowerPoint 필요
  • 파일이 사용 중입니다
  • 지원되지 않는 파일 형식
  • 결과 파일을 저장할 수 없습니다

이렇게 오류 원인을 바로 이해할 수 있도록 구분해, 사용자가 다음 행동을 빠르게 결정할 수 있게 했습니다.

실행 환경 및 주의 사항

  • Windows 10 / 11 환경에서 실행해야 합니다.
  • Microsoft PowerPoint 정식 설치 버전이 필요합니다.
  • 변환 전에 PowerPoint를 완전히 종료하는 것을 권장합니다.
  • 변환 대상 PPT 파일이 열려 있지 않아야 합니다.
  • 배포용 문서는 원본 대신 복사본으로 작업하는 것을 권장합니다.
  • 문서는 외부 서버에 업로드하지 않고 로컬 환경에서 처리합니다.

이런 분들께 특히 유용합니다

  • 고객사 전달용 PPT를 준비하는 분
  • 회의실 PC나 행사장 PC에서 발표해야 하는 분
  • 폰트 저작권이나 레이아웃 깨짐 이슈를 줄이고 싶은 분
  • 작업용 원본과 배포용 파일을 분리해 관리하는 팀
