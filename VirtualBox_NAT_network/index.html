VirtualBox Net network 사용하기

가상 환경에서 설정 > 네트워크 > NET 네트워크 를 선택하면 이름이 없다는 오류가 난다. VirtualBox의 환경설정 > 네트워크 > 추가 > 이름 설정 을 한후, 가상 환경에서 설정 > 네트워크 > NET 네트워크 를 선택하면 이름을 선택할 수 있다.

추가적으로 위와 같이 설정을 하면 호스트에서 가상으로 접근이 되지 않는다.

가상 환경의 설정에서 2번째 네트워크를 추가 하고 호스트 전용 어댑터를 선택한다. ifconfig 명령어로 호스트의 가상 IP 대역을 확인하고, 다음 명령어를 이용하여 가상서버의 호스트 전용 네트워크를 설정한다. 호스트의 IP 값이 192.168.53.100 일 경우 다음과 같이 설정한다.

$ sudo vi /etc/network/interfaces
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

source /etc/network/interfaces.d/*

# The loopback network interface
auto lo  
iface lo inet loopback

# The primary network interface
auto enp0s3  
iface enp0s3 inet dhcp

# The host-only network interface 신규로 추가
auto enp0s8  
iface enp0s8 inet static  
address 192.168.53.200  
netmask 255.255.255.0  
network 192.168.53.0

이후 네트워크를 다음의 명령어로 재시작 한다.

$ sudo systemctl restart networking
or  
$ sudo /etc/init.d/networking restart

참고 자료를 확인하면 자세한 그림과 설명이 되어 있다.
