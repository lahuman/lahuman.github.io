[2017-09-21] IntelliJ 를 사용하자

21 Sep 2017

Reading time ~1 minute

IntelliJ가 좋다고 하여 사용해보기를 몇번 시도를 했지만, 늘 실패 하였다.

일단 유료이기도 하고, 무료 버젼에서는 WAS와의 연동으 지원 하지 않기에, 사용이 불편하였다.

물론 요즘은 대부분 SpringBoot를 이용한 개발을 진행하여, Community 버젼을 이용하여도 충분하다.

하루 정도 사용해보고 불편한 점은 Eclipse에 익숙하여 그런지 UI 적응이 안되는 것과, 단축키이다.

다행이 인터넷에 Eclipse 사용자를 위한 IntelliJ 단축키 이미지가 있다.

이런건 가볍게 출력하여서 책상에 붙여 놓고 쓰다보면 손에 익을 꺼라 생각한다.

톰켓 설정 하기

WAS를 설정 하는 일은 조금 당황 스럽다. SERVER를 추가 하는 것이 아니라 RUN 설정에서 톰켓을 추가 하여야 한다.

우선 다음 그림과 같이 Edit Configurations.. 를 클릭한다.

그리고 아래 그림에서 +를 누르면 여러 종류의 실행 가능한 방식이 표출 된다 이중에 Tomcat Server을 선택 하면 Local과 Remote가 표시된다.

Local을 누르면 Tomcat Server가 추가 되고 설정을 해야 한다. Context를 추가 하려면, 두번째 탭인Deployment를 선택 하고 + 버튼을 클릭 후, Artifact... 버튼을 클릭 한다.

선택을 하면 아래 그림과 같이 팝업이 뜨는데, 시작이 같은 이름의 프로젝트가 2개 뜬다. exploded가 붙은 프로젝트를 선택하면 된다.

마지막으로 Application 의 Context 명만 설정하고 저장하면 설정은 끝난다.

구동은 Alt+Shift+F10를 누르면 선택이 나오거나, 첫번째 그림에서 화살표 표시를 누르면 된다.

앞으로 몇일이나 쓸까?
