WSL 에서 JAVA SPRINGFRAMEWORK를 VSCODE로 구동해보자

VS Code에서 WSL Remote환경을 이용하여 Java개발 환경 구성하기에 정리가 잘되어 있어서 따로 추가 작성하지는 않습니다.

사용해본 경험만 작성하면 다음과 같습니다.

강점

  • 정말 가벼운 개발 환경
  • maven, debuging 지원

약점

  • 가볍기만한 개발환경
    • 개발할때 손이 많이 간다.
  • 초기 구축시 손이 많이감

결론

현재 사용하는 노트북이 i5 cpu에 4GB 메모리를 가지고 있습니다. IntelliJ나 Elipse를 사용하기에는 살짝쿵 메모리가 부족한데 간단한 개발 한다고 하면 추천 합니다.
