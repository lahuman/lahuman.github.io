Mac M1에서 docker로 mysql 실행

Mac M1에서 mysql을 실행하기 위해서 많이 검색을 해보았지만, 실제 동작되는 건 아래 명령어입니다.

$ docker run -d -p 3306:3306 --name mysql -e MYSQL_ROOT_PASSWORD=<password> mysql/mysql-server:latest-aarch64

여러가지를 찾아봤지만, mysql/mysql-server:latest-aarch64 이 이미지가 확실하게 돌아가네요!
