Ubuntu 20.04] npm install sqlite3 설치시 오류 처리

WSL에서 Ubuntu 20.04가 제공된다는 이야기를 듣고 바로 설치 하였습니다.

node를 설치하고 진행중인 개인 프로젝트를 설치 하였는데, npm install sqlite3에서 다음과 같은 오류페이지가 표출 되었습니다.

오류를 처리하기 위해서 많은 시도를 하였지만, 모두 실패 하였습니다.

결국 최후의 방법으로 Ubuntu를 삭제하고 재설치를 진행하였습니다.

다음과 같이 처리하여 문제를 해결하였습니다.

# ubuntu update
$ sudo apt-get update
$ sudo apt-get -y upgrade

# 개발도구 설치
$ sudo apt-get install -y build-essential

# node 12 설치
$ curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
$ sudo apt-get install -y nodejs

$ sudo npm install sqlite3

아주 잘 설치됩니다.

거의 2주간 여러 라이브러리를 설치하고 지우고를 했는데, 결국 포멧후 첨부터 다시하니 쉽게 되네요.

역시 최후의 방법은 포멧입니다.
