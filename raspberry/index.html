raspberry pi 3 를 얻었다!

09 Jul 2019

Reading time ~1 minute

raspberry pi 3 초기 설정

황부장님에게 Raspberry Pi 3 Model B을 얻어서 설치와 설정을 해보았다.

사실 이번에 Raspberry Pi 4가 나와서 개인적으로 구매하여 PC 처럼 사용해 보려고 했다. 일단 Raspberry Pi 4를 현재는 구하기도 어렵고, 국제 배송을 해야 하기 때문에 오래 걸린다.

먼저 라즈베리파이3 B+ 설치가이드(2) RASPBIAN OS 설치를 보고 Raspbian OS를 설치 하였다. 크게 어렵지는 않지만 다운로드에 오랜 시간이 걸린다.

한글 설정은 nabi, fcitx를 이용해서 테스트 하였지만 잘 안되었다.

Raspberry PI : 한글 환경 설정, 한글 입력기(uim) 설치를 보고 uim으로 설치 하니 문제 없이 되었다.

VSCODE와 VIM을 IDE로 설치하고, nodejs까지 설치하여 설정은 마무리 하였다.

다만 이것 저것 설정을 하다보니, 온도가 뜨거워 진다는 메시지를 확인할수 있었다.

몇일 써보고 정말 괜찮으면 Raspberry Pi 4를 구매해야겠다.
