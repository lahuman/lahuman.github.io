예시) WEB 개발자 모집 공고

얼마전 팀원분의 요청으로 WEB 개발자 모집 공고를 작성하였다.

적당히 여기 저기서 긁어 모아서 읽어본 후, 정리를 하였는데 읽어 보면 그럴듯 하다.

[수행업무]

  • 관제 업무 시스템과 솔루션 개발을 진행
  • 추가 개발은 dhtmlx와 spring framework를 이용하여 개발 예정
  1. 주요 업무
    • 보안 솔루션 관련 개발 업무
    • SPA 기반의 UI 개발
    • REST API 및 외부시스템 연동
  2. 직무 자격요건
    • 해당 분야 3년 이상의 경험 또는 그에 준하는 능력
    • 소스관리(Svn/Git/…)에 대한 이해/경험
    • 성능 튜닝 경험
    • 다양한 환경을 고려한 개발 경험
    • Java/Spring/ORM 기술 기반에서의 API/Back-end 개발에 대한 익숙함
    • Data Modeling 경험

[자격 요건]

  • 학력 : 무관
  • 경력 : 관련경력 3년 이상
  • 사용기술 : javascript, java, spring, dhtmlx, maven, gulp 등

[공통 자격요건]

  • 팀원과 깊은 신뢰를 형성하고 유지해 나가는데 노력을 아끼지 않으시는 분
  • 구성원이 이끌어 나가는 팀 문화를 만들 분
  • 새로운 것, 모르는 것을 두려워 하지 않으며 배운 것을 공유하는 것을 즐기시는 분
  • 자기개발에 노력을 기울이고 팀원들과 같이 성장해 나가실 분

[우대사항]

  • 자료구조/알고리즘/디자인패턴/리팩토링에 대한 깊은 이해와 활동 경험
  • Full-stack 개발 경험
  • CI 적용 및 테스트 자동화 경험
  • 오픈 소스 발전에 이바지한 경험
