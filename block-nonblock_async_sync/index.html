  • caller : 호출 하는 함수

  • callee : 호출 당하는 함수

  • blocking :
  • non-Blocking :

동기 Non-blocking

비동기 Non-blocking

sequenceDiagram; Caller->>Callee: call; Callee->>Caller: return void; note right of Callee: getResult; note right of Callee: callback;

정리

  동기 비동기
Blocking caller는 아무것도 할수 없는 상태가 되며, 결과를 얻은 후 직접 처리 caller는 아무것도 할수 없는 상태가 된며, 결과는 caller가 처리
Non-Blocking caller는 자기 할 일을 할 수 있고, 결과를 얻은 후 직접 처리 caller는 자기 할 일을 할 수 있고, 결과는 callee가 처리
