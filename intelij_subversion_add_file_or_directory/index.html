IntelliJ에서 추가된 디렉토리가 Version Control System에 추가 되지 않을때 처리 방법

회사 동료 분의 도움으로 IntelliJ 적응 중이다.
이클립스에서는 쉽게 되었던 몇몇 부분이 인텔리제이에서는 잘 되지 않는다.

그중 VCS(Version Control System)에 디렉토리를 추가 하는 것이 아무래 해도 안되었다.

원인은 svn:global-ignores에 전체(*)가 걸려 있어서 였다.

삭제하는 방법은 다음과 같다.

먼저 해당 프로젝트에서 마우스 오른쪽 키 > Subversion > Set Property...를 선택 한다.

그러먼 다음과 같이 Set Property에 대한 팝업이 뜬다.

여기서 svn:global-ignores를 선택 한다

하단에 나오는 문자열 중에 * 문자열을 삭제 한다.

삭제후 다음과 같이 표출되면 OK 버튼을 클릭 한다.

이후 Ctrl + K 를 누르면 디랙토리를 추가 할 수 있다.
