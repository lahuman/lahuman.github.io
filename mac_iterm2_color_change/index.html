mac에서 iTerm2의 디렉터리 색상 바꾸기

mac에서 zsh을 설치 하지 않고 단순하게 directory 색상만 변경하고 싶다. vi로 개발을 하는 것도 아니고 zsh는 많은 기능이 제공되어 나아게 필요 없다.

처음에 시도한 방법은 테마를 이용하여 처리 하려고 했다.

하지만 테마를 바꾼다고 하여도 실제 디렉터리의 색상이 변경 되는 것은 아니었다.

그러다 찾은 문의 글에서 LSCLORS를 알게 되었다.

사용방법은 단순하게 .bash_profile 에 다음 라인을 추가 하면 된다.

$> cd ~
# .bash_profile 파일이 없으면 새로 생성 한다.
$> vi .bash_profile
export LSCOLORS="EHfxcxdxBxegecabagacad" 

# 이부분이 ls 시 색상을 입힌다.
alias ls='ls -GH'  
# ll을 많이 사용하는 편이라서 추가 했다.
alias ll='ls -alGH'

.bash_profile 팡일이 없어서 잠시 당황을 했지만 그냥 생성하면 처리된다.
