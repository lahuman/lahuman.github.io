4. Daily-Quest

01 Sep 2023

Reading time ~1 minute

나만 쓰는 서비스이지만, 열심히 업뎃 하자!

쿼스트에 점수(Point)와 퀘스트의 담당자(Member)를 추가 하였습니다.

담당자 - Member

담당자를 등록 하고, 담당자의 총 포인트를 확인 할 수 있습니다. 담당자의 삭제 기능은 복잡함 때문에 제외처리 했습니다.

점수 - Point

Quest 등록시 담당자와 포인트를 등록하는 항목이 추가 되었습니다. 포인트의 경우 음수/양수로 입력이 가능하고 음수의 경우 빨간색, 양수의 경우 파란색으로 표기 됩니다.

해당 값들은 옵션으로 입력하지 않으면 포인트는 0으로 표기 됩니다.

마치며

아이들의 퀘스트 관리용으로 잘 사용하고 있습니다. 혼자 쓰는게 조금 아쉽지만 재미있게 사용하고 있네요!
